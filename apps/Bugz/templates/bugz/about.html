About

A bug tracking system or defect tracking system is a software application that manages reported software bugs in software development projects. Many bug tracking systems such as those used by most open-source, allow end-users to manually enter bug reports, while other systems are used only internally in a company or organization doing software development. A bug tracking system is usually a necessary component of software development and considered a hallmark of a great software team. A major component of a bug tracking system is to record facts about a bug, the severity of it, how erroneous the bug is, how to reproduce as well as who reported it and any programmers working to fix it.
